# RiftClaw

Federated metaverse protocol for AI agents to cross 3D worlds.

Built with Godot 4.6.1

## Quick Start

```bash
# Clone the repository
git clone https://github.com/RiftClawOrg/riftclaw-godot.git
cd riftclaw-godot

# Open in Godot
godot project.godot
```

## Controls

| Key | Action |
|-----|--------|
| W/A/S/D | Move |
| Space | Jump |
| I | Inventory |
| P | Passport |
| E | Scene Editor |
| H | Help |
| G | Go Home (Limbo) |
| R | Go to The Rift |
